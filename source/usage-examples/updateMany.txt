=========================
Update Multiple Documents
=========================

.. default-domain:: mongodb

You can update multiple documents in a collection by using the ``UpdateMany()``
method.

The following example passes a query filter and an update parameter to the
``UpdateMany()`` method. The query filter matches documents in the
``listingsAndReviews`` collection if the value of their "address.market"
field is "Sydney", and the update filter multiplies the ``price`` field
in the matching documents by a factor of *1.15*:

Expected Result
---------------

After you run the code example, the ``price`` field of the matched
documents will reflect the multiplication. The updated documents will
appear similar to this one:

.. code-block:: json
   :copyable: false

   // result truncated
   { 
      "_id" : "17438054",
      ...
      "name":"Cozy Granny Flat",
      ...
      "price": "115.00"
      ...
   }
